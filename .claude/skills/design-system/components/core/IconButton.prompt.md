Icon-only button for toolbars, table rows, and headers — always supply `label`.

```jsx
<IconButton label="Settings"><GearIcon/></IconButton>
<IconButton variant="solid" label="Copy"><CopyIcon/></IconButton>
<IconButton variant="accent" label="Add"><PlusIcon/></IconButton>
```

Variants: `ghost` (default), `solid` (raised), `accent` (clay tint → fills on hover). Sizes sm/md/lg.
