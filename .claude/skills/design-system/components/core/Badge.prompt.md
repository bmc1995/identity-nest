Status / metadata pill in JetBrains Mono — for connection states, environments, counts.

```jsx
<Badge tone="success" dot>active</Badge>
<Badge tone="warning">expiring</Badge>
<Badge tone="accent" solid>production</Badge>
<Badge tone="neutral">OIDC</Badge>
```

Tones: neutral / accent / success / warning / danger / info. `solid` for filled, `dot` for a leading status dot.
