User or org avatar. Squircle by default (matches the app-icon language); initials are auto-colored from the name.

```jsx
<Avatar name="Dana Reyes" status="online" />
<Avatar src="/u/dana.jpg" name="Dana Reyes" size="lg" />
<Avatar name="Acme Inc" round size="sm" />
```

Sizes xs/sm/md/lg. `round` for circular, `status` for a presence dot.
