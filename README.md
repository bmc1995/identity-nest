## Description

### (**WIP**)

A [Nest](https://github.com/nestjs/nest) server handling authentication and authorization

The end goal of this project is to achieve [certification](https://openid.net/certification/) of conformance with the [OpenID Connect Core 1.0](https://openid.net/specs/openid-connect-core-1_0.html) specification, which adds a simple identity layer on top of the [OAuth 2.0](https://datatracker.ietf.org/doc/html/rfc6749) protocol.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
